# chatgpt_sdgs

Classify texts by SDG using ChatGPT.

The prompt does not include examples for each SDG. It relies on ChatGPT's
existing knowledge of the SDGs.

Response is in JSON format.

## Parameters



- `model`: ChatGPT model to use
